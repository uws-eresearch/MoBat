Member Function Documentation

AbstractRepository::constructObject (   $id = NULL)
abstract

This method is a factory that will return a new repositoryobject object that can be manipulated and then ingested into the repository.

Parameters
string$idThe ID to assign to this object. There are three options:
  • NULL: An ID will be assigned.
  • A namespace: An ID will be assigned in this namespace.
  • A whole ID: The whole ID must contains a namespace and a identifier in the form NAMESPACE:IDENTIFIER
Returns
AbstractObject Returns an instantiated AbstractObject object that can be manipulated. This object will not actually be created in the repository until the ingest method is called.
