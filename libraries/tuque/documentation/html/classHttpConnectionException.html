Detailed Description

HTTP Exception.

This is thrown when a status code other then 2XX is returned.

Parameters
string$messageA message describing the exception.
int$codeThe error code. These are often the HTTP status codes, however less then 100 is defined by the class extending HttpConnection, for eample cURL.
array$responseThe array containing: status, headers, and content of the HTTP request causing the error. This is only set if there was a HTTP response sent.
